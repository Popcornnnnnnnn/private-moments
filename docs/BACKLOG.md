# Private Moments Backlog

这个文件记录暂不进入当前开发主线、但后续可能有价值的想法。进入正式开发前再补充需求细节、交互方案和验收标准。

## 隐私锁

状态：暂缓

背景：Moments 是私人生活记录工具，后续如果内容变多、使用频率变高，可以考虑增加应用级隐私保护。

可能功能：

- Face ID / 设备密码解锁：打开 App 或重新进入前台时先验证，通过后才显示内容。
- 后台遮挡：App 进入后台或切换任务时显示遮罩，避免多任务界面暴露时间线内容。
- 锁定策略：支持每次打开都锁、离开一段时间后锁、或关闭隐私锁。

暂缓原因：当前阶段优先完善核心记录、浏览、整理和同步能力，隐私锁不是马上阻塞使用的问题。
